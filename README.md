# chat-app

-   Run 'npm run build' to install all dependencies and build frontend
-   Run 'npm start' to start the server
-   Run 'npm run start-back' to run backend in development mode
-   Run 'npm run start-front' to run frontend in development mode

### Create a .env file in backend folder with the following content (Necessary for backend to work)

PORT={Your desired port}  
MONGO_URI={MongoDb connection uri}  
JWT_KEY={Password for jwt. Whatever value you want}  
TENOR_KEY={Your tenor api key}
